Git

Ini tentang Git

Ada 9 macam git

Berikut adalah macam

git init : untuk menganalisis repositori

git config : untuk konfigurasi username dan email

git status : untuk mengecek status

git add : untuk menambahkan file

git commite -m : untuk memberikan commit / mencatat perubahan

git clone : untuk mengambil repositori dari server web

git push : untuk mengirimkan repositori ke server

git pull : untuk menarik commit dari server


Branch

Ini tentang Branch

Ada 6 macam branch

git branch : untuk melihat branch

git branch all : untuk melihat semua nama branch

git branch nama_branch : untuk membuat branch

git checkout nama_branch : untuk berpindah branch

git -d nama_branch : untuk menghapus

git checkout nama_branch : untuk berpindah branch
